#### Installation

- > nvm use 14+
- > npm i

#### Development

- > gulp

#### Add fonts

- > if otf font convert to ttf
- > add ttf fonts to src/fontss
- > gulp
- > uncomment @import "fonts" from src/scss/_index.scss

#### Start server on the 4200 port

- > gulp/tasks/server.js -> port: 4200
- > http://localhost:4200/
